## How to Run

1. Install Dart SDK from [Dart's official website](https://dart.dev/get-dart).
2. Clone this repository:
   ```bash
   git clone https://github.com/ARShishir/LearnDart.git
   ```
3. Navigate to the project directory:
   ```bash
   cd All About Dart With Motalab vai
   ```
4. Run any Dart file:
   ```bash
   dart main.dart
   ```
